## 添加截图说明

如果你想为项目添加实际的截图：

1. 创建 `docs/screenshots/` 目录
2. 访问部署的网站
3. 使用截图工具捕获各个场景
4. 将图片保存到 `docs/screenshots/` 目录
5. 更新此文档中的图片链接

推荐的截图工具：
- macOS: Cmd + Shift + 4
- Windows: Snipping Tool 或 Snip & Sketch
- Linux: GNOME Screenshot 或 Flameshot
- 浏览器扩展: Full Page Screen Capture
